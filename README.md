# SMART-INDUSTRIAL-PRODUCT-DETECTION-PCB-
A compact and reliable industrial PCB designed for real-time product counting, monitoring, and alerting. This system is ideal for production lines, packaging units, and warehouse tracking where accurate counting and immediate fault notification are critical. If the system detects no product movement, it automatically sends sms.

FEATURE AND APPLICATIONS :  
1.) Configurable SMS Alert Interval
Selectable time intervals: 5 min, 15 min, 30 min, 60 min, 120 min
Flexible alert timing based on production needs
2.) Multi-User Notification Support
Supports up to 5 mobile numbers
Ideal for roles like Operator, TL, Manager, Head, GM
3.) Wireless Number Management
Add / Delete / Replace phone numbers remotely
No physical access required
4.) Data Logging & Display
Product count data exportable to Excel
Real-time count display via OLED screen
5.) Data logging of Scrap products by manual button pressing by operator

CONDITIONS : 
1.) Sensor Compatibility
Supports only 24V Industrial PNP Photoelectric Sensors
Designed specifically for reliable industrial detection
2.) Control Interface
Operates using only 3 physical buttons:
Scrap Button – for rejecting / marking defective count
Reset Button – to reset product count
Config Mode Button – to enter configuration setting
